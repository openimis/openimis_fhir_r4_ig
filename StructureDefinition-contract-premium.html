Extension: Premium (Contract)

Indicates a premium for a Contract. This is a complex extension.

The official URL for this extension is: 

https://openimis.github.io/openimis_fhir_r4_ig/StructureDefinition/contract-premium

Status: active
Extension maintained by: Swiss TPH

Context of Use

This extension may be used on the following element(s):

  • OpenIMISContract

Usage info

Usage:

  • Use this Extension: openIMIS Contract
  • Examples for this Extension: Contract/InformalSectorContract
